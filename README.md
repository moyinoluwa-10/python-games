# Python Games

A collection of simple Python games.

## Games

### Blackjack
A classic card game where the goal is to get a hand value as close to 21 as possible without going over.

### Countdown to 21
A countdown game where you try to reach 21.

## Getting Started

Each game is contained in its own directory. To run a game, navigate to its folder and execute the main script.

```bash
# Run Blackjack
python blackjack/

# Run Countdown to 21
python countdownto21/game.py
```

## Requirements

- Python 3.6+
